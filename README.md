# Netflix Landing Page Clone

## 📌 Project Overview
This project is a frontend case study and structural replication of the Netflix landing page. It was developed to apply core web development concepts, focusing on responsive design, modern UI/UX principles, and CSS layout techniques. 

## 🚀 Features
* **Hero Section:** A visually striking header with a background image overlay, integrated Google Fonts, and the classic email subscription call-to-action.
* **Feature Highlights:** Recreated content blocks (e.g., "Enjoy on your TV") with clean alignment.
* **Trending Now Gallery:** A horizontal gallery utilizing CSS Flexbox to display movie posters uniformly without breaking the page structure.
* **Frequently Asked Questions:** A cleanly structured layout mimicking the accordion-style FAQ section.

## 🛠️ Technologies Used
* **HTML5:** Semantic structuring of the web page.
* **CSS3:** Custom styling, Flexbox for complex layouts, hover effects, and external font integration.

## 📚 Academic Context
This project was built as a practical application of UI design and web architecture principles for the Fundamentals of Web Programming.
